*Sobre la pagina*

Poké-Buscador es una aplicación frontend moderna que actúa como una enciclopedia digital para las criaturas de las regiones clásicas (Gens 1-4). El objetivo principal del proyecto fue crear una interfaz intuitiva que permitiera el consumo de datos complejos de una API externa, presentándolos en un formato minimalista y profesional.

Puntos clave de la aplicación:
Experiencia de Usuario (UX) Fluida: Gracias a la integración con la PokéAPI, la página permite realizar búsquedas en tiempo real, ofreciendo una respuesta inmediata sin necesidad de recargar el sitio.

Diseño Visual Temático: La interfaz utiliza una paleta de colores vibrantes basada en la identidad de la franquicia, combinada con un fondo de estilo pixel-art que refuerza la temática de los videojuegos originales.

Fichas de Datos Detalladas: Cada Pokémon cuenta con una tarjeta de información expandida que desglosa sus tipos mediante etiquetas de colores, sus habilidades únicas y estadísticas base (HP, XP, peso y altura), facilitando una lectura rápida y estructurada.

Arquitectura Limpia: El proyecto sigue una estructura de archivos organizada (index, css, js), lo que demuestra buenas prácticas de desarrollo, escalabilidad y facilidad de mantenimiento.

Enfoque en la Responsividad: La disposición de las tarjetas en cuadrícula se adapta automáticamente a diferentes tamaños de pantalla, garantizando que la experiencia de búsqueda sea igual de cómoda en un computador de escritorio o en un dispositivo móvil.
